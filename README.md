# Airplane Queries - SQL

A relational database project for managing airline data. Includes database creation scripts and a set of SQL queries covering various airline management scenarios.

## Features

- Database schema creation with tables for flights, passengers, employees, and more
- Data insertion scripts to populate the database
- Queries covering filtering, joining, grouping, and aggregating airline data

## Technologies

- Language: SQL
- Database: SQL Server / MySQL

## How to Run

1. Open your SQL client (SQL Server Management Studio, MySQL Workbench, etc.)
2. Run `database_script.sql` first to create and populate the database
3. Run `Final Exam Airline Queries.sql` to execute the queries
